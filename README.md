# CWW - Conpago Weather Watch

## Setup

### Server
1. cd server
2. npm install
3. copy example.env and create .env, fill with OpenWeatherMaps API Key
4. Install nodemon using `npm i -g nodemon` (not included in devDeps as often bloats CI/CD)
5. npm run dev

### Frontend
1. cd webapp
2. npm install
3. npm run frontend
